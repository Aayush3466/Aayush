Publications & Research Works

Peer-reviewed papers, manuscripts under review, and independent research investigations across cybersecurity, bioinformatics, quantum ML, and AI systems.

01 IEEE Published IIT Mandi, India
Enhancing DDoS Attack Detection: A Hybrid SVM-Decision Tree Ensemble Approach
The 15th International IEEE Conference on Computing, Communication and Networking Technologies (ICCCNT)
Developed a hybrid machine learning model combining SVM and Decision Trees to detect DDoS attacks. Achieved very high accuracy on the CICDDoS2019 dataset, enhancing network security through an ensemble methodology that leverages the complementary strengths of both classifiers.
↗ View Paper
02 IEEE Published IIT Mandi, India
Predicting DDoS Attacks: A Machine Learning Approach using ALDDoS Dataset
The 15th International IEEE Conference on Computing, Communication and Networking Technologies (ICCCNT)
Utilized machine learning models including RandomForest and XGBoost to predict DDoS attacks with very high accuracy using the ALDDoS dataset. Enhanced cyber defense capabilities by developing a custom web proxy policy as a preventive layer.
↗ View Paper
03 Under Review Results in Engineering · arXiv
A Two-Stage Ensemble Feature Selection with Particle Swarm Optimization for Microarray Data Classification in Distributed Computing Environments
Submitted to Results in Engineering · Available as arXiv preprint
Proposed an ensemble feature selection method leveraging Particle Swarm Optimization for microarray data classification. Improved classification performance in distributed computing environments by selecting optimal gene subsets — addressing the high-dimensional challenge inherent in bioinformatics datasets.
Aayush Adhikari, Sandesh Bhatta, Harendra Singh, Amit Mishra, Khair Ul Nisa, Abu Taha Zamani, Aaron Sapkota, Debendra Muduli, Nikhat Parveen
↗ arXiv Preprint
04 Manuscript Complete
A Quantum-Enhanced Ensemble Learning Approach for Accurate Classification of Functional and Non-Functional Requirements in Software Engineering
Manuscript · Submission Pending
Developed a quantum-enhanced ensemble learning framework for software requirements classification, achieving 95.05% accuracy by integrating variational quantum circuits with classical machine learning and transformer models — a novel hybrid paradigm at the intersection of quantum computing and NLP.
Sandesh Bhatta, Aaron Sapkota, Aayush Adhikari, Debendra Muduli, Jyoti Bhushan Dahal
↗ View Manuscript
05 Individual Research
Deep Learning-Based Detection and Classification of Nepali Currency Denominations Using Optimized CNNs
Independent Research Work
Built a deep learning system using MobileNetV2 for real-time classification and counterfeit detection of Nepali currency denominations. Optimized for edge-device efficiency, making the solution viable for deployment in resource-constrained environments.
